# PRODIGY_SD_01

## Software Development Internship Task 01  
### Prodigy Infotech

## Task Description
Build a Temperature Conversion Program that converts temperatures between
Celsius, Fahrenheit, and Kelvin.

## Features
- Takes temperature value as input
- Accepts unit (C / F / K)
- Converts temperature into the other two units

## Technology Used
- Java
- Scanner class

## How to Run the Program
1. Compile the program:
   javac TemperatureConverter.java
2. Run the program:
   java TemperatureConverter
3. Enter temperature value and unit when asked

## Sample Input
25 C

## Sample Output
Temperature in Fahrenheit: 77.0  
Temperature in Kelvin: 298.15
